# OCR Document Processing System

This project is an OCR-based document processing system built using Python, Tesseract OCR, OpenCV, and Streamlit.

## Features
- Extracts text from invoice images
- Identifies Invoice Number, Date, and Amount
- Supports PNG, JPG, JPEG formats
- Saves extracted data into files
